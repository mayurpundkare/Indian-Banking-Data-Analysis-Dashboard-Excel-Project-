# **🏦 Indian Banking Data Analysis Dashboard (Excel Project)**

# **📌 Project Overview**

The Indian Banking Data Analysis Dashboard is an interactive Excel-based analytics project designed to analyze banking transactions, 
customer activities, revenue trends, and fraud insights. The dashboard transforms raw banking data into meaningful visual reports, 
helping stakeholders monitor performance and make data-driven decisions.

The project demonstrates practical skills in Microsoft Excel, data cleaning, pivot tables, charts, dashboard design, and business reporting.

# **🎯 Project Objective**

The primary objectives of this project are:

- Analyze banking transaction data efficiently.

- Monitor revenue and transaction trends over time.

- Track customer and regional performance.

- Identify fraud patterns and suspicious transactions.

- Build an interactive dashboard for business decision-making.

- Convert raw data into meaningful visual insights.

# **📊 Dashboard Insights**

The dashboard provides valuable business insights such as:

✅ 500 banking transactions were analyzed in the dashboard.

✅ ₹93.93 Million+ worth of transactions were processed.

✅ 500 unique customers are included in the analysis.

✅ 25 bank branches across 5 regions were analyzed.

✅ Only 13 transactions (2.6%) were marked as fraudulent.

✅ 434 transactions (86.8%) were completed successfully.

✅ 26 transactions (5.2%) failed during processing.

✅ 19 transactions (3.8%) are in pending status.

✅ 21 transactions (4.2%) were reversed.

✅ The dashboard enables analysis of transaction types, payment modes, customer activity, regional performance, and fraud detection through interactive visualizations.
